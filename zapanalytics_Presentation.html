Your use case
System architecture
Users & SAP
End users hit the system from outside Azure; the client's on-prem SAP connects separately via RFC calls (tRFC/qRFC/bgRFC) over ExpressRoute.
Entry point
Nginx proxy in Azure fronts all traffic and routes into the Kubernetes cluster.
Core app
ZapAnalytics runs in a Docker container inside K8s, with PostgreSQL alongside it as the database.
AI layer
ZapAnalytics integrates with an Agentic AI component backed by Azure OpenAI.
Reporting
Data flows out to Power BI for dashboards.
Pattern
Hybrid setup — on-prem SAP as source of data, Azure-hosted containerized ZapAnalytics with AI Agents, Power BI for consumption.
Security & Data Privacy
On-Premises or in your cloud
All data remains within the organization's infrastructure at all times, ensuring full control and maximum security. No processing at Zapliance.
Read-Only SAP Access
ZapAnalytics accesses the SAP system for data extraction via RFC using read-only permissions — no accidental deletions.
Azure ExpressRoute Private Path between SAP and Azure
Creates a private, non-internet path that securely carries SAP RFC traffic between on-premises and Azure.
NGINX gateway
NGINX proxy server centralizes all browser traffic with SSL/TLS encryption, so one certificate securely protects all container access.
Information Security Management System (ISMS) Overview
We run a strong ISMS that ensures consistent security and reliability across all operations and services.

Standardized Procedures VS Agentic AI
Autonomy
The system makes decisions independently or significantly influences human decisions.
Business Relevance
Its outputs affect real organizational outcomes, not just routine transaction processing.
Learning-Based
The way the machine makes decisions is not merely based on static rules (e.g. "if-then-else chaining").
Dynamic Adaptability
The system can continuously improve and adapt as new data is collected and retraining occurs.
Specificity
It is a customized AI solution tailored to a particular use case, not a generic off-the-shelf tool.
